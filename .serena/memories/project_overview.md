# Project Overview
Mantis is a web application with a FastAPI backend and a Next.js frontend, using PostgreSQL and Keycloak.

## Tech Stack
- **Backend**: Python, FastAPI, Poetry, Alembic, SQLAlchemy
- **Frontend**: TypeScript, Next.js, Tailwind CSS, pnpm
- **Database**: PostgreSQL
- **Auth**: Keycloak
- **Infrastructure**: Docker, Docker Compose

## Structure
- `fastapi_backend/`: Python backend code
- `nextjs-frontend/`: TypeScript frontend code
- `docs/`: Documentation
- `docker-compose.yml`: Container orchestration
